# archive/

旧データ・移行作業・調査ファイルの保管場所。
本番コードには含まれない。git 管理下に置くことで履歴を保持する。

## ディレクトリ構成

```
archive/
└── 取込/          移行作業・インポート関連ファイル
    ├── *.csv      景品発注履歴CSV、在庫インポート用CSVなど
    ├── *.xlsx     景品棚卸・在庫移行作業用 Excelファイル
    ├── *.json     PCH/SGP抽出データ
    ├── *.py       移行スクリプト（Python）
    ├── *.js       インポートスクリプト（Node.js）
    ├── *.html     移行レビュー用HTMLツール
    └── IMG_*.JPG  参照用写真
```

## 注意

- このディレクトリのファイルは **本番ビルドに含まれない**（vite.config.js の対象外）
- 機密情報が含まれる可能性があるため、公開リポジトリにプッシュする前に内容を確認すること
- GASレガシーファイルは存在しない（2026-04-20 調査済み）
